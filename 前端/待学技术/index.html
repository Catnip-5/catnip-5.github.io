前端待学技术

发表于 1997-01-01 更新于 2025-04-23 分类于 前端 阅读次数：

框架：
Vue、React：首选
Next.js：全栈
Svelte：性能
Angular：企业级
编程语言：
JavaScript (ECMAScript)：小型项目（说明：ECMAScript是规范，是JavaScript 的标准，新特性）
TypeScript：大型、复杂项目，标配
样式方案：
Tailwind CSS：快速开发
SCSS：传统项目
PostCSS：CSS 优化
CSS-in-JS (Styled-Components, Emotion)：React 项目组件化动态样式，（说明：CSS-in-JS技术范式，Styled-Components 和 Emotion 是其实现）
数据请求：
Axios：基于 Promise 的 HTTP 库，跨框架或复杂请求
Fetch API：浏览器内置的 HTTP 请求 API，对比Axios更轻量且无依赖，替代 XMLHttpRequest
React Query：React 项目复杂数据管理
SWR：React 项目简单数据获取
构建工具：
Vite：新项目，开发体验和速度优于 Webpack
Webpack：复杂或老项目
esbuild：性能优先，但需其他工具配合
服务：
Node：运行时环境
代码规范：组合使用
ESLint：代码检查工具
Prettier：自动格式化代码
全局状态管理：
Redux：大型 React 项目
Zustand：中小型 React 项目
Pinia：Vue 3 标配，取代 Vuex
测试与工具：
Vitest：基于 Vite 的单元测试框架，取代 Jest
Cypress：易用的 E2E 测试，调试体验优秀
Playwright：强大的 E2E 测试，跨浏览器支持

组件库与 UI 框架：
Material-UI (MUI)：React 的 UI 组件库
Ant Design：React 的企业级 UI 组件库
Chakra UI：React 的轻量 UI 组件库
Headless UI：自定义 UI，Tailwind 配合
Element Plus：Vue 3 企业后台
Vuetify：Material 风格 Vue 应用
动画与交互：
Chart.js：图表库
GSAP：动画库
Framer Motion：React 专用的动画库
Three.js：3D库
D3.js：SVG 操作数据，专注 2D 数据可视化
国际化：
i18next：JavaScript 国际化框架
React Intl：React 专用的国际化库
性能优化与监控：
Lighthouse：Google 提供的 Web 性能和 SEO 审计工具
Sentry：错误监控和性能追踪平台

待学顺序：
JavaScript (ECMAScript): 这是前端的基石，先打好基础。
HTML5 和 CSS3: 学习网页结构和样式。
任选一个前端框架 (Vue 或 React): 建议先掌握一个框架的核心概念。
TypeScript: 在掌握 JavaScript 基础上学习 TypeScript。
样式方案 (Tailwind CSS, SCSS, PostCSS): 学习 CSS 预处理器和工具。
数据请求 (Axios 或 Fetch API): 学习如何从服务器获取数据。
构建工具 (Vite 或 Webpack): 学习如何打包和构建你的代码。
代码规范 (ESLint, Prettier): 学习如何保持代码风格一致。
Next.js: 在掌握 React 基础上学习 Next.js。
