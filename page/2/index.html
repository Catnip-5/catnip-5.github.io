JDK17：新特性

发表于 2025-04-24 更新于 2025-04-27 分类于 JDK

正式特性

Foreign Function & Memory API（外部函数和内存）

  • 合并孵化模块Foreign-Memory Access API 和 Foreign Linker API。
  • 提供标准化的堆外内存和本地函数调用接口。
  • 安全的堆外内存操作（MemorySegment, MemoryAddress）。
  • 纯 Java 的本地函数调用，替代 JNI，性能接近 JNI，安全性更高。
  • 孵化版本：
    • JDK 14：引入 Foreign-Memory Access API，提供 MemorySegment、MemoryLayout 等核心功能，支持本地内存操作，替代 Unsafe。
    • JDK 15：优化 API，引入 MemorySession，增强布局、语义与生命周期控制。
    • JDK 16：引入 Foreign Linker API，支持本地函数调用；增强 MemorySession（支持并发），优化布局和性能，稳定 API 设计。
